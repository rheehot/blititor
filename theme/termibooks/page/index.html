라즈베리파이 노드제이에스

노드JS를 활용한 라즈베리파이 실습

GPIO

다양한 GPIO 모듈을 통해 하드웨어를 콘트롤할 수 있습니다. 각종 센서와 컨트롤러에 통신을 수행하기 위해 GPIO 핀을 조작하는 과정을 배웁니다.

Linux

라즈베리파이에는 리눅스 데비안 기반의 라즈비안 운영체제가 설치됩니다. 리눅스 터미널 환경을 공부하며 노드JS 코드를 실행하여 프로젝트를 진행할 수 있습니다.

Web Page

노드JS 는 현재 가장 흥미로운 웹 애플리케이션 서버로 유명합니다. 파일 시스템과 데이터베이스에 자료를 저장하고 웹 페이지를 통해 데이터를 출력합니다.
