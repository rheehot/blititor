Cocos Creator로 당신의 첫 번째 게임을 만들어보세요

Application

박현천

KOSSLAB 4 기 전담
튜토리얼
내용
오픈소스 크로스플랫폼 게임엔진인 Cocos2d-x 기반의 통합 에디터인 코코스 크리에이터를 이용하여 함께 게임을 만들어보는 과정을 경험해보려고 합니다. 계속 점프를 뛰며 가속도가 붙는 몬스터 캐릭터를 조작하여 화면 상에 나타나는 별을 모으는 간단한 캐주얼 게임을 만들어볼 예정으로 리소스 준비부터 씬 편집과 스크립트 작성까지 단계별로 모든 과정을 함께 진행합니다. 사용언어는 Javascript이며 프로그래밍 초심자도 크게 어렵지 않게 따라오실 수 있으실 정도의 수준이며 노트북에 미리 코코스 크리에이터를 설치해서 오셨으면 좋겠네요.

내용
리엑트 네이티브를 사용하여 Javascript로 iOS, Android 네이티브 어플리케이션을 만들어 보는 과정입니다. 외부 API를 이용해 코인 시세를 읽어와 데이터를 가공하여 보여줄 것입니다. javascript es6, npm 사용에 대한 기본적인 지식이 필요하고 준비 사항으로는 Expo를 사용하여 프로젝트를 생성하고 시뮬레이터 상에 `Hello World`를 출력해보는 과정을 진행해보시고 오면 됩니다. 이번 튜토리얼에 사용할 github repository를 참조하세요. (https://github.com/JeffGuKang/ReactNative-Tutorial)
